# 지움AI 첫 사용자 안내서 v1.8

작성자: 22B Labs · 제4의 길 (The 4th Path)

메타 설명: 지움AI를 처음 쓰는 피해자와 보호자가 당황하지 않도록 돕는 1분 안내서입니다.

레이블/태그: 지움AI, 첫사용자, 피해자안내, 개인정보, 삭제요청, 디지털성범죄, 로컬우선, 22B Labs

---

## 먼저 숨을 고르세요

지금 필요한 것은 모든 정보를 완벽하게 쓰는 일이 아닙니다.

처음에는 네 가지만 있으면 충분합니다.

1. 어디에 올라왔는지
2. 무엇이 노출됐는지
3. 언제 발견했는지
4. 지금 협박이나 위험이 있는지

지움AI는 피해물을 대신 지우는 앱이 아닙니다. 대신 당신이 공식 경로로 움직일 수 있도록 사건을 정리하고, 요청서와 상담 준비자료를 만들어 줍니다.

---

## 급한 경우

다음 중 하나라도 해당하면 앱 입력보다 사람의 도움과 안전 확보가 먼저입니다.

- 지금 협박을 받고 있음
- 돈, 만남, 추가 사진·영상 요구가 있음
- 성적 이미지, 불법촬영물, 딥페이크, 몸캠피싱이 의심됨
- 미성년자 피해 또는 미성년자 시기 촬영물이 관련됨
- 가족, 학교, 직장에 퍼뜨리겠다는 말이 있음

이 경우에는 `112`, `1366`, 중앙디지털성범죄피해자지원센터, 경찰청 ECRM, 방송미디어통신심의위원회 신고 경로를 우선 확인하세요.

지움AI 안에서는 "긴급한 유포/협박 피해예요"를 고르면 됩니다.

---

## 절대 적지 마세요

지움AI는 아래 정보를 요구하지 않습니다.

- 비밀번호
- 주민등록번호
- 카드번호
- 신분증 원본 이미지
- 피해 이미지나 영상 원본 파일
- 인증번호, 복구코드, OTP

전화번호나 이메일이 사건 설명에 필요할 수는 있지만, 가능하면 "전화번호가 노출됨", "이메일이 노출됨"처럼 종류만 적으세요.

---

## 첫 화면에서 무엇을 고르면 되나요

상황 문장을 기준으로 고르면 됩니다.

| 이런 상황이면 | 첫 화면에서 고를 것 |
|---|---|
| 전화번호, 주소, 이름, 얼굴 사진이 올라갔어요 | 개인정보가 노출됐어요 |
| 예전에 내가 쓴 글이나 사진을 지우고 싶어요 | 예전 게시물을 지우고 싶어요 |
| 원본은 지웠는데 검색 결과에 계속 떠요 | 검색 결과에 계속 떠요 |
| 남이 올린 내 사진이나 이미지 삭제를 요청하고 싶어요 | 사진/이미지를 직접 삭제 요청하고 싶어요 |
| 딥페이크, 불법촬영, 성적 이미지, 유포 협박이 있어요 | 긴급한 유포/협박 피해예요 |

잘 모르겠으면 가장 가까운 것을 고르세요. 지움AI는 선택지를 바탕으로 안전한 첫 경로를 다시 잡습니다.

---

## 무엇을 적으면 되나요

처음 입력은 짧아도 됩니다.

좋은 예:

```txt
제 얼굴 사진과 이름이 커뮤니티 게시글에 올라갔어요.
URL은 알고 있고, 게시자는 닉네임만 보입니다.
협박은 아직 없습니다.
```

더 안전한 예:

```txt
전화번호와 학교 정보가 노출됐습니다.
게시 위치와 발견 일시는 알고 있습니다.
주민등록번호나 비밀번호는 적지 않겠습니다.
```

위험한 예:

```txt
제 비밀번호는 ...
주민등록번호는 ...
피해 영상을 첨부합니다.
```

위험한 예처럼 쓰면 지움AI는 결과 생성을 멈춥니다. 이것은 오류가 아니라 안전장치입니다.

---

## 삭제 권한 질문은 왜 있나요

"내 사진"과 "내가 그 페이지를 지울 권한"은 다릅니다.

그래서 지움AI는 삭제 권한을 따로 묻습니다.

- 내가 올린 게시물: 직접 삭제 가능성이 있습니다.
- 내가 관리자인 페이지: 직접 삭제 가능성이 있습니다.
- 피해자의 동의를 받은 대리인: 요청은 도울 수 있지만 직접 삭제는 조심해야 합니다.
- 사진 속 당사자이지만 게시자는 다른 사람: 직접 삭제가 아니라 삭제 요청 경로로 가야 합니다.
- 권한이 불명확함: 삭제 요청서와 공식기관 연결만 제공합니다.

이 구분은 악용을 막기 위한 장치입니다. 피해자를 돕는 도구가 타인의 게시물, 리뷰, 증거, 공익 제보를 지우는 도구가 되면 안 됩니다.

---

## 결과 화면은 이렇게 읽으세요

결과 화면은 보통 네 덩어리입니다.

1. 지금 바로 할 일
2. 삭제 요청서 또는 상담 준비자료
3. 접근경로 증거목록
4. 공식기관과 재확인 계획

처음에는 "지금 바로 할 일"만 따라가도 됩니다.

문서를 복사하기 전에는 한 번만 확인하세요.

- 내가 적지 말아야 할 비밀번호나 주민등록번호가 들어갔는가
- 피해물 원본을 보내라는 내용이 있는가
- 삭제 권한이 없는데 직접 삭제처럼 보이는 문장이 있는가
- 공식기관이나 플랫폼에 직접 제출할 준비가 되었는가

---

## 저장은 꼭 해야 하나요

아닙니다.

공용 PC, 학교 컴퓨터, 회사 컴퓨터, 가족이 함께 쓰는 기기라면 저장하지 않는 편이 안전합니다.

저장하면 현재 브라우저에만 남습니다. 서버로 올라가지는 않지만, 같은 브라우저를 쓰는 사람이 볼 수 있습니다.

정확한 URL 보관도 꼭 필요할 때만 켜세요. 기본값은 URL 경로를 숨기는 쪽입니다.

---

## 끝났다고 바로 믿지 마세요

삭제됐다고 보여도 다시 확인해야 합니다.

- 7일 후 같은 검색어로 확인
- 30일 후 다시 확인
- 90일 후 마지막 확인
- 다시 올라오면 기존 기록을 덮어쓰지 말고 새 접근경로로 추가

삭제 완료와 사용자 직접 확인 완료는 다릅니다. 지움AI가 이 둘을 나누는 이유가 여기에 있습니다.

## 한 문장 결론

지움AI는 당신에게 더 많은 설명을 요구하는 도구가 아니라, 덜 드러내고도 다음 행동을 할 수 있게 돕는 도구입니다.

천천히, 적게 쓰고, 공식 경로로 움직이면 됩니다.
