# 지움AI 법률·형사 지원 서비스 연계 설계 v1.2

메타 설명: 피해자가 비용을 크게 쓰기 전에 공식·무료 경로와 필요한 법률 상담을 안전하게 연결하는 지움AI 연계 설계.

레이블/태그: 법률지원, 형사신고, 디지털성범죄, 온라인피해365센터, ECRM, KISA, 법률구조, 로컬우선, 22B Labs

## 왜 이 기능이 필요한가

피해자가 가장 먼저 잃는 것은 시간이 아니라 판단력입니다. 유출, 협박, 사칭, 명예훼손, 개인정보 침해가 한꺼번에 닥치면 어디에 신고해야 하는지조차 흐려집니다.

지움AI의 역할은 대리인이 되는 것이 아니라 길을 밝혀주는 것입니다. 자동 제출, 자동 고소, 유출자 단정은 하지 않습니다. 대신 사건을 분류하고, 제출 전 문서를 만들고, 공식·무료 경로부터 민간 법률 플랫폼까지 안전한 순서로 연결합니다.

## 연계 원칙

1. 공식·무료 경로를 먼저 보여줍니다.
2. 유료 법률 플랫폼은 “선택형 상담 채널”로만 표시합니다.
3. 피해 URL, 민감정보, 피해물 원본은 외부 서비스로 자동 전달하지 않습니다.
4. 고소, 신고, 삭제요청, 법률상담 신청은 사용자가 최종 확인한 뒤 직접 제출합니다.
5. 유출자 특정은 게시자 단서 정리까지만 지원하고, 신원 확인은 수사기관·법원·플랫폼 절차에 맡깁니다.

## 반영한 공개 서비스

| 서비스 | 성격 | 지움AI 안에서의 연계 방식 |
|---|---|---|
| 중앙디지털성범죄피해자지원센터 | 공식·무료 | 디지털성범죄 사건의 최우선 경로. 상담, 삭제지원, 유포 모니터링, 수사·법률·의료 연계로 연결 |
| 경찰청 ECRM | 공식·무료 | 경찰 신고 준비서를 복사해 사이버범죄 사전 접수와 경찰서 방문 준비로 연결 |
| KISA 개인정보침해 신고센터 | 공식·무료 | 개인정보 노출·침해 사건의 신고·상담 경로로 연결 |
| 온라인피해365센터 | 공식·무료 | 피해 유형이 복합적이거나 소관 기관이 불명확할 때 1:1 피해구제 상담 허브로 연결 |
| 방송미디어통신심의위원회 | 공식·무료 | 플랫폼 삭제 요청만으로 부족한 불법·권리침해 정보의 삭제·접속차단 심의 경로로 연결 |
| 대한법률구조공단 | 공공 법률지원 | 무료 또는 저비용 법률상담, 법률구조 가능성 확인으로 연결 |
| 대한변협 나의 변호사 | 공개 법률 플랫폼 | 형사·민사 전문 변호사 검색과 상담 선택지로 연결 |
| 로톡, 로앤굿 | 민간 법률 플랫폼 | 빠른 유료 상담, 고소대리, 경찰조사 동행 등 선택지로 연결하되 비용 확인을 명시 |

## 앱 구현 방식

- `lib/publicResources.ts`: 사건 유형별 외부 서비스 디렉터리와 연계 방식을 정의합니다.
- `lib/responsePack.ts`: 사건 분류 결과에 맞는 법률·형사 서비스 목록을 대응 패키지에 포함합니다.
- `components/ResponsePackPanel.tsx`: “법률·형사 지원 서비스 연계” 섹션을 보여주고 외부 이동 전 확인창을 띄웁니다.
- `components/ResourceRouter.tsx`: 결과 화면의 기관 추천을 공식·공공·민간 법률 플랫폼으로 구분합니다.
- `lib/export.ts`: 내보내기 문서에 서비스별 링크, 준비물, 연계 방식, 개인정보 주의사항을 포함합니다.

## 남은 과제

- 각 서비스의 공개 API 또는 제휴 API가 생기면 서버 opt-in 모드에서만 어댑터를 붙입니다.
- 피해자가 거주 지역을 입력했을 때 지역 디지털성범죄피해자지원센터를 우선 보여주는 로컬 라우팅을 추가할 수 있습니다.
- 변호사 상담 연결은 광고·추천처럼 보이지 않도록, 비용·이해충돌·전문분야 확인 체크리스트를 계속 강화해야 합니다.
