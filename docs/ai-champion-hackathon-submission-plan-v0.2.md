## 4. 발표 핵심 메시지

1. 피해자는 유포 경로가 아니라 조각난 화면을 들고 옵니다.
2. 담당자는 관찰 근거와 추정 신호가 섞이면 판단이 느려집니다.
3. 지움AI는 사건을 시간순 그래프로 바꾸고, 기관이 확인할 질문을 만듭니다.
4. 피해물 원본, 비밀방 잠입, 가해자 신원 단정은 하지 않습니다.
5. 본선 4시간 안에는 Graph export와 합성 데모 케이스를 구현 목표로 잡습니다.

## 5. 본선 4시간 구현 목표

1. 0~40분: 합성 케이스 1~3개 고정
2. 40~110분: 추격 다이어그램 Graph JSON/Markdown export 구현
3. 110~170분: 기관 인계 요약 카드 생성
4. 170~220분: 안전 거절 테스트와 UI 정리
5. 220~240분: 발표 데모 리허설

핵심은 새 AI 모델을 만드는 것이 아니라, 현재 작동하는 추격 엔진에 제출 가능한 산출물을 붙이는 것입니다.
